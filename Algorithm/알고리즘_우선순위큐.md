# 우선순위 큐
- 추가되는 순서와 상관업시 값이 작은 원소 반환
```python
from queue import PriorityQueue
q = PriorityQueue()
# q = PriorityQueue(maxsize=8) 최대 크기 지정 가능

# 원소 추가
q.put()
# 원소 삭제
q.get()
# 출력
print(q.get()) # 인덱스 접근 불가, non iterable, for문 불가
# 길이
q.qsize()
# 비거나 다 찼거나
q.empty()
q.full()
# 정렬기준 변경
print(q.get()[1]) # 몇 번째 기준으로 오름차순 정렬할 것인지 지정
```

### 시간복잡도: O(log n)
