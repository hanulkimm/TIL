## Bootstrap Grid System
### Bootstrap grid system
- 요소들의 디자인과 배치에 도움을 주는 시스템
- flexbox로 제작
- 기본 요소
  - Column: 실제 컨텐츠를 포함하는 부분
  - Gutter: 칼람과 칼럼 사이의 공간 (사이 간격)
  - Container: column들을 담고 있는 공간
- container, rows, column으로 컨텐츠를 배치하고 정렬
- 12개의 column, 6개의 grid breakpoints
